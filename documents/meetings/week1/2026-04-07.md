## ✅ 결정 사항
- **장바구니**: 전체 상품 일괄 주문으로 구현 (간단 구현)
- **쿠폰 생성**: 관리자가 직접 생성, 스케줄러로 만료 처리
- **타임세일**: 특가 타입으로 관리, 종료 시 상품 상태 → 판매종료
- **주문 취소**: 구현 X (주문 아이템이 모두 삭제되면 주문 자동 취소 처리)
- **인기 검색어**: 10분 단위 업데이트, 동일 사용자 동일 키워드 1회 카운트, 00시 초기화
- **상품 조회**: 전체 상품 조회 + 특가 상품 별도 조회, 정렬 기준 최신순
- **쿠폰 수량**: total_count만 관리, 0이 되면 발급 종료
- **쿠폰 정책**:
    - 할인 방식: 할인 금액으로 통일 (할인율 X)
    - 일반 상품: 2만원 이상일 때 쿠폰 사용 가능
    - 특가 상품: 쿠폰 사용 불가
    - 혼합 주문 (일반 + 특가): 특가 상품이 하나라도 포함되면 쿠폰 사용 불가
    - 특가 상품: 2만원 이상일 때 구매 가능
- **타임세일**: 하나의 상품은 하나의 타임세일만 가능 (1:1 관계, UNIQUE 제약조건)
