## 🧩 핵심 검증 포인트

### 1. 동시성 검증

- 쿠폰 발급 → Redis 락 + DB 조건 검증
- 주문 재고 → Redis 락 + DB 조건 검증

### 2. 데이터 정합성

- 재고 음수 방지
- 쿠폰 중복 발급 방지
- 주문 후 상태 일관성 유지

### 3. 인증 / 권한

- JWT 인증 검증
- 관리자 권한 검증
